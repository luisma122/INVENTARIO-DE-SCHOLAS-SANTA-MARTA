¡Error de Permisos de Firebase!

Tu base de datos está bloqueando el acceso. Sigue estos 3 pasos para arreglarlo:

  1. Ve a la Consola de Firebase.
  2. Entra en Firestore Database > pestaña Rules (Reglas).
  3. Borra lo que haya y pega el siguiente código:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

⚠️ Nota: Una vez pegado, dale clic al botón azul que dice "Publish" (Publicar). Luego recarga esta página.
